
Einige kurze Tipps zu sitecopy unter Windows
--------------------------------------------

Die offizielle Webseite von Sitecopy-Win ist leider schon einige Zeit nicht
mehr erreichbar. Daher von mir einige kurze Tipps. Den genauen Syntax der
Anweisungen zur Steuerung von sitecopy entnehmt ihr bitte der Datei

	/man/man1/sitecopy.1.html

Diese Referenz gilt jedoch für Linux-Systeme. Bei Windows sind einige
Dinge gesondert zu beachten.

1. Installation
---------------

Installieren sie die Datei sitecopy-cygwin-setup.exe

sitecopy.exe sollte nun unter dem Pfad C:\Programme\sitecopy zu finden
sein.

Sitecopy besitzt keine grafische Benutzeroberfläche sondern wird über
die Kommandozeile gesteuert.

Starten sie die Windows Eingabeaufforderung über
START - Programme - Zubehör - Eingabeaufforderung

Wechseln sie in das Verzeichnes von Sitecopy
(cd C:\Programme\sitecopy)

geben sie sitecopy ein und drücken die Eingabetaste

Sollten sie nun eine Fehlermeldung erhalten, dass irgendwelche DLLs
fehlen, kopieren sie diese bitte aus dem Ordner fehlende-dlls
nach C:\Programme\sitecopy

2. Konfigurationsdateien
------------------------

Sitecopy benötigt die Konfigurationsdatei .sitecopyc (mit Punkt am Anfang)
in der die Zugangsdaten und Einstellungen für das Spiegeln eines FTP-Servers
festgelegt werden. Eine Beispieldatei ist beigefügt. Der genaue Syntax steht
in der /man/man1/sitecopy.1.html
Beispieldatei des Programm-Autors unter /share/examplerc
Die Datei .sitecopyc müssen sie vor dem ersten Start mit einem Texteditor
erstellen und unter C:\Programme\sitecopy\.sitecopyc speichern

Zudem wird durch Sitecopy eine XML-Datei mit dem Namen der FTP-Seite erstellt, 
die Informationen über die Dateien und Verzeichnisse des FTP-Servers enthält.

Sitecopy sucht diese Dateien normalerweise im Root-Verzeichnis des Benutzers.
Da Sitecopy ursprünglich ein Linux-Programm ist, werden die Dateien evtl. im
falschen Pfad gesucht. Daher sollten sie den Pfad beim Starten von Sitecopy
über die Kommandozeile unter Windows immer mit angeben.

Beispiel für eine Fetch-Site Anweisung mit Pfadangaben zu .sitecopyc:
sitecopy --rcfile=C:\Programme\sitecopy\.sitecopyc --storepath=C:\Programme\sitecopy\ -fo microsoft
(alle Dateien und Verzeichnisse des FTP-Servers werden ermittelt)

Beispiel für eine Synchronize-Site Anweisung:
sitecopy --rcfile=C:\Programme\sitecopy\.sitecopyc --storepath=C:\Programme\sitecopy\ -s microsoft
(Dateien werden auf die lokale Festplatte synchronisiert)
